Cara Membuka Hasil
  1. Klik Unduh Hasil Excel di bawah — file akan otomatis terunduh ke komputer Anda.
  2. Buka file PAPI_Kostick_[Nama]_Hasil.xlsx menggunakan Microsoft Excel atau Google Sheets.
  3. Sheet "Skor" menampilkan 20 skala dengan skor, persentase, dan interpretasi otomatis.
  4. Sheet "Grafik" menampilkan bar chart profil kepribadian lengkap.
  5. Sheet "Summary" menyajikan laporan ringkas dan panduan interpretasi.
